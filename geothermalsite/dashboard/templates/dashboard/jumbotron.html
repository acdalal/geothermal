Welcome to GeoDE!

A Carleton Geothermal Data Explorer developed by the Geothermal CS Comps Team, 2022-23

Disclaimers

One of the difficulties (and joys!) of working with Carleton's Geothermal data is its size and complexity. Here are a few things to know:

  • This database contains many outages, so certain searches will contain gaps. Find out more about data outages.
  • This database is big, so searching for data over longer periods of time will, well, take longer! Some may take a matter of seconds, others minutes. Please be patient!
  • This is an ongoing project, so there are many features we hope are implemented in the future.
