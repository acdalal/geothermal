Welcome to GeoDE!

A Carleton Geothermal Data Explorer developed by the Geothermal CS Comps Team, 2022-23

Disclaimers

Carleton's geothermal database is large and complex! Here are a few things to know:

  • This database contains many outages, so certain searches may have unexpected contours. Find out more about data outages.
  • This database is big, so searching for data over long periods of time may take a few seconds and up to a few minutes. Please be patient!
  • This is an ongoing project and there are many features we hope are implemented in the future! Read more about future directions.
