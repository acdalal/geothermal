About the Project

Our task was to build an interface that interacts with and displays the geothermal data for the broader Carleton community. We were advised by Prof. Amy Csizmar Dalal. To learn more about the background of the project, visit the Carleton computer science comps page.

We are very hopeful that GeoDE can provide value to members of the Carleton community well into the future! We know that there are lots of exciting ways that this site can be expanded upon, and we encourage those who are interested to read more about future directions and improvements for GeoDE on our repository.

About the Geothermal System

Carleton's geothermal system is part of Carleton's utility master plan, which includes a goal of being carbon-free in 2050. The data on this website pertains to the geothermal system that was spearheaded by the Carleton Aquifer Research Lab (CARL) and installed on campus in 2017-2018. The geothermal system contains three interconnected borefields which serve to power Carleton's cooling and heating system. These borefields act as a "ground source heat pump" by transferring excess heat to and from Carleton's fluid circulation. They are located on the Bald Spot, the Mini-Bald Spot, and Bell Field. The Bald Spot is the largest system and it has 134 vertical boreholes that each run 525 feet deep. The Mini-Bald Spot has 77 vertical boreholes, that each run 520 feet deep. Bell Field has 47 “double-stacked” horizontal boreholes, that each run for 510 ft in length. All three of these systems (Bald Spot, Mini-Bald Spot, Bell Field) converge in the East Energy Station, located in the basement of Anderson Hall. Fiber-optic cables were installed in 5 of the 134 boreholes on the Bald Spot, and the data collected by these cables is what is stored in the database.

About the Data Pipeline

The database is maintained by Bruce Duffy, Technical Assistant of the Physics department. For the current data collection configuration, measurements from each of the 5 boreholes are collected in every 30 minutes. The overview of the data pipeline for the geothermal system is as follows: the 5 fiber optic cables embedded in the Bald Spot boreholes converge at a switchbox in a utility tunnel north of the Chapel, which is connected to a Silixa XT-DTS unit in a network closet in the Chapel basement. The XT-DTS is a ruggedized distributed temperature sensor unit, and collects data from the boreholes by sending laser pulses down the fiber optic loops. It sends the data files it generates to a private PC via ethernet network, and these data files are uploaded to a Linux Virtual Machine that captures the relevant data and stores the database.
